# Codex Starter Agents

Этот файл описывает стандартные роли субагентов, которые используются в Codex Starter.

## Roles

- `researcher` — исследователь: поиск по коду, документации и интернет-источникам, сбор фактов.
- `implementer` — исполнитель: разработка кода, исправление багов, добавление тестов.
- `reviewer` — рецензент: проверка качества, code review, поиск рисков.

## Как использовать

1. Определите задачу.
2. Подберите роль субагента по типу работы.
3. Запустите субагента и дождитесь результатов.
4. Менеджер фиксирует вывод, коммитит и обновляет `SNAPSHOT.md`.
