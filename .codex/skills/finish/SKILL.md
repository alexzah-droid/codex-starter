# Skill: Finish Session

## Что делать

1. **Проверить состояние:**
   - прочитать `.codex/SNAPSHOT.md`
   - проверить `git status`
   - зафиксировать прогресс в лог файле

2. **Запустить тесты:**
   ```bash
   ./scripts/run-tests.sh 2>/dev/null || true
   ```
   или эквивалентный запуск `npm test`, `pytest` и т.д.

3. **Обновить SNAPSHOT.md:**
   - добавить результат тестов
   - отметить завершённые задачи
   - обновить статус и дату

4. **Сделать краткий отчет:**
   - что сделано
   - что осталось
   - найденные проблемы

## Чего НЕ делать

- Не публиковать детали конфиденциальных данных
- Не удалять логи и файлы без причины
- Не продолжать работу, пока не зафиксирован итог текущей сессии
